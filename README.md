# ESP32_S2_PICO

.NET nanoFramework 演示 微雪的 ESP32-S2-Pico 使用

# 案例说明

1. SerialPortHello： 串口打印 hello
2. BlinkBoardLed：板载 RGB 控制，WS2812
3. RGBLed：外接全彩 RGB 灯的控制
